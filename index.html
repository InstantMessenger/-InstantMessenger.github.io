О МЕССЕНДЖЕРЕ

Мессенджер INSTANT создается на Python PySide6 и QML QtQuick. Это некоммерческий проект - мы развиваем его ради саморазвития и получения собственного опыта ;)

СИСТЕМА КОНТРОЛЯ ВЕРСИЙ

Мы используем Git в связке с GitHub для эффективной командной работы. Это позволяет нам делиться изменениями, параллельно работать над разными задачами и сохранять всю историю разработки проекта в одном месте

ТЕХНОЛОГИИ РАЗРАБОТКИ

Для интерфейса используем QML (Qt Meta Language) и JavaScript. Такой стек обеспечивает нам кроссплатфорaменность и удобство разработки. Серверную часть пишем на Python с FastAPI + Websocket

НАДО ПОНИМАТЬ
